# 実行前チェックリスト

このチェックリストは `AGENTS.md` の要点を、実行前やレビュー前に短く確認したいときの参照用にまとめたものです。ここに新しい規範を追加せず、正本の更新が必要なら `AGENTS.md` を修正します。

## Pre-Flight

- [ ] 変更を伴う作業なら `main` / `master` 以外で進めている
- [ ] 秘密情報や `.env` を露出する操作が含まれていない
- [ ] ローカル探索で足りる事実を先に確認した
- [ ] 現在性や高リスク判断がある場合は一次情報確認の要否を判断した
- [ ] 新機能の開発なら、先行テスト方針またはテスト駆動開発（TDD）を外す理由を決めている
- [ ] 破壊的操作や無関係な巻き戻しをしない方針が守られている

## During Execution

- [ ] 進捗共有は短く、必要なときだけ行っている
- [ ] 技術事実は説明文ではなく現在のコード、設定、テストで確認している
- [ ] 複雑なタスクでだけチェックリストや協働プレイブックを使っている
- [ ] 略語を使うときは正式名または平易な日本語を先に書いている

## Before Finish

- [ ] 実行したテストや未実施項目を明記できる
- [ ] 新機能でテスト駆動開発（TDD）を外した場合は、理由と代替検証を明記できる
- [ ] 外部調査が判断に影響した場合は、正本 docs か設計判断メモに要点を残した
- [ ] 参照切れや古い導線を新たに増やしていない
