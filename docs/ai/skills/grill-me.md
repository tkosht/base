# grill-me

- Purpose: 計画や設計を 1 問ずつ掘り下げ、分岐ごとの決定を詰める
- Role: 全方位の深掘り用
- Use when: 計画の stress test、設計の詰め、`grill me` / `grill-me` と明示されたとき
- Alternative: `grill-me-essential-first` は本質優先の設計詰め用。重要論点が閉じるまで細部を `parking lot` に送る
- Codex entrypoint: `.agents/skills/grill-me`
- Claude entrypoint: `.claude/skills/grill-me`
- Upstream source: `mattpocock/skills/grill-me`
- Sync command: `make sync-skill SKILL=grill-me`
