# Security Policy

このテンプレートは公開・private の両方で使えるように作られています。

## Reporting

- 公開 repo では、実際の連絡先・security advisory フローへ更新してください。
- private repo では、社内のインシデント窓口へ更新してください。

## Template Defaults

- 実 secret はコミットしない
- `.env*` と `secrets/**` は protected path として扱う
- shared MCP / agent settings はレビュー済みのものだけをコミットする
